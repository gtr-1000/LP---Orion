# ORION — Landing Page Delivery Document

---

## 1. Strategic Foundation

**Framework chosen: PAS → How It Works → Hormozi Value Stack → Social Proof → Low-Risk Offer**

For a **warm SaaS audience**, classical AIDA wastes screen real estate educating people who already know they have a problem. The correct play is to **agitate the pain immediately, bridge to the solution fast, then overwhelm with value before revealing price.** By the time the user sees "$29/mo," they've already mentally calculated the alternative costs ($2K+/month agencies, hours wasted) — the price feels laughable.

The Hormozi-style Value Stack is critical for this price point: small business owners are price-sensitive, so you need to reframe the value *before* the price lands.

---

## 2. Page Architecture

| # | Section | Strategic Objective |
|---|---------|-------------------|
| 1 | **Sticky Nav** | Always-visible CTA — captures decision-ready visitors at any scroll depth |
| 2 | **Hero** | Outcome-first headline + immediate free trial CTA. 3 social proof stats below the fold to anchor credibility |
| 3 | **Logo Bar** | Instant authority transfer — warm audiences need trust signals early |
| 4 | **Pain Section** | Agitate the 3 core pains (time, money, competition). Creates "they get me" effect |
| 5 | **How It Works** | Destroy the "this is complicated" objection. 3 steps = simplicity signal |
| 6 | **Features as Benefits** | Each feature framed as an outcome, not a spec. Eliminates "so what?" reactions |
| 7 | **Testimonials** | Specific numbers (312% ROAS, 51% CAC cut) beat vague praise 100% of the time |
| 8 | **Value Stack** | Anchors value at $4,896/yr before revealing $29/mo price. Creates massive perceived gap |
| 9 | **Pricing** | Featured middle plan is the conversion target. Ghost CTAs drive all to free trial |
| 10 | **FAQ** | Handles top 5 objections silently — no sales rep needed |
| 11 | **Final CTA** | FOMO-anchored headline ("competitors are already using AI") + ring visual creates urgency |

## 3. Conversion Optimization Notes

**A/B Tests to Run (Priority Order):**

1. **Hero Headline** — Test outcome-led ("Stop Running Campaigns") vs. fear-led ("Your Competitors Are Already Using AI") vs. curiosity-led ("What If Your Marketing Ran Itself?")
2. **Primary CTA Text** — "Start Free — No Credit Card" vs. "Try ORION Free for 14 Days" vs. "Get My Free Campaigns"
3. **Value Stack position** — Test moving it *above* pricing vs. current position
4. **Hero social proof** — Test swapping the 3 stats for real user avatars + star rating (warmer, social feel)
5. **Pricing anchor** — Test showing annual pricing with a "save 30%" badge

**Friction Points to Watch:**
- "No credit card" must be visible on the first CTA and the final CTA — never hide it
- The value stack uses real market price comparisons — keep these updated as competitors adjust pricing
- FAQ should grow as your actual sales objections evolve (use Gong/call data)

**Recommended Visual Elements for Production:**
- Hero: animated dashboard mockup or short Lottie loop showing campaign creation
- How It Works: simple screen recording GIFs per step (nothing converts like "show don't tell")
- Testimonial section: real photos outperform initials avatars by ~22% (Baymard Institute data)
- Final CTA section: consider a subtle animated particle background in brand color

**Design Notes:**
- Color palette: near-black `#07070C` + electric lime `#C8FF00` — avoids the clichéd purple gradient 99% of SaaS tools use
- Typography: **Syne** (bold, geometric, technical authority) + **DM Sans** (readable, modern body)
- Scroll-triggered fade-up animations on all cards — reduces cognitive load, guides attention
- Sticky nav captures visitors at all scroll depths without interrupting UX
